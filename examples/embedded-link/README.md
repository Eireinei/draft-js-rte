# Embedded link

## How to use
Download the example or [clone the repo](https://github.com/romainleduc/draft-js-rte):

```sh
curl https://codeload.github.com/romainleduc/draft-js-rte/tar.gz/master | tar -xz --strip=2 draft-js-rte-master/examples/embedded-link
cd embedded-link
```

Install it and run:

```sh
npm install
npm start
```
or:

[![Edit on CodeSandbox](https://codesandbox.io/static/img/play-codesandbox.svg)](https://codesandbox.io/s/draft-js-rte-embedded-link-oh2be)

## The idea behind the example

This example demonstrates how you can use embedded link using draft-js-rte hooks.
