# Font size

## How to use
Download the example or [clone the repo](https://github.com/romainleduc/draft-js-rte):

```sh
curl https://codeload.github.com/romainleduc/draft-js-rte/tar.gz/master | tar -xz --strip=2 draft-js-rte-master/examples/font-size
cd font-size
```

Install it and run:

```sh
npm install
npm start
```
or:

[![Edit on CodeSandbox](https://codesandbox.io/static/img/play-codesandbox.svg)](https://codesandbox.io/s/draft-js-rte-font-size-ulgfs)

## The idea behind the example

This example demonstrates how you can use font size using draft-js-rte hooks.
